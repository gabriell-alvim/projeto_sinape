📦 Dossiê de Montagem de Documentação

Busca as exigências e a pasta do processo no Painel Sinape, organiza os documentos do SharePoint em pastas letradas e devolve um .zip pronto.

Este servidor roda só na sua máquina. Antes do primeiro uso, copie config.exemplo.json para config.json e preencha as credenciais do Azure AD e o token do Painel. O clique acima chama o Graph API de verdade e baixa os arquivos binários originais — não é uma prévia nem uma cópia em imagem.
